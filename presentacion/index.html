Un disco para instalar

Un disco...

... o un pendrive.

 

GECOS v2 se instala en 10 minutos, dejando el PC listo para entregar al usuario final.

 

Y se puede instalar cientos, miles, millones de veces sin multiplicar los costes.

 

Es Software Libre

1:Instalar

La instalación inicial es muy sencilla.

 

Descargar la imagen del disco de instalación, grabarla en un DVD o pendrive, arrancar el ordenador con él y seguir los pasos del Asistente de Instalación.

2:Vincular

Para gestionarlo a distancia conectaremos el nuevo puesto al Centro de Control GECOS y a un sistema de identificación de usuarios (LDAP o Directorio Activo).

 

Para eso ejecutamos el Asistente de Configuración.

¿Quiere verlo paso a paso?

1:Instalar

  • Descargue el disco de GECOS
  • Pase el archivo .ISO a un DVD o un pendrive e insértelo en el nuevo puesto.
  • Compruebe que el ordenador inicia desde el lector de DVD o desde el puerto USB

 

Espere y...

y pulse Continuar

... que el equipo tiene red

... que está enchufado (si es un portátil)

 

y pulse Continuar

Pulse Continuar una última vez

¡Fin!

2:Vincular

G.C.A.

o GECOS Config Assistant, es el programa que le guiará paso a paso para vincular el puesto al Centro de Control

 

Sólo puede ejecutarlo un superusuario, como el que hizo la instalación inicial.

La red

es imprescindible.

 

Compruebe que su conexión ha sido detectada correctamente.

 

Pulse Siguiente

Ahorre tiempo

dando los parámetros de configuración en el Centro de Control, y descargándolos en el puesto.

 

Así sólo deberá recordar una URL.

La hora exacta

es fundamental para que puestos y servidores confíen unos en otros.

 

Proporcione la dirección de un servidor de tiempo (ntp) al que los puestos puedan acceder.

Identifique

Vincular

También puede tener puestos GECOS independientes.

Avance

sin detenerse en esta pantalla, los datos se rellenan automáticamente.

 

No marque la casilla si es un puesto nuevo.

¿LDAP

o Directorio Activo?

 

Un puesto GECOS puede validar usuarios utilizando cualquiera de los dos sistemas.

 

O puede crear usuarios sin validación externa.

Usuarios

para soporte técnico, para emergencias o en casos donde no existe un sistema de validación centralizado.

 

Los puede añadir ahora, o mejor aún, salte este paso y hágalo desde el Centro de Control.
